# Hospital Administration System (Console Version)
This project is a simple **Hospital Management System** built using **Python** and **SQLite3**.
It provides console-based management for:
- Patients
- Doctors
- Appointments
- Billing & Payments
---
## 􍠽􍳌 Features
### 􍠽􍱨􂀍􂚕􏸏 Patient Management
- Add new patients
- List all patients
- Search patients by ID or name
### 􍠽􍱨􂀍􂚕􏸏 Doctor Management
- Add new doctors
- List all doctors
### 􍠽􍳅 Appointment Management
- Schedule new appointments
- List all appointments with doctor & patient name
### 􍠽􍲵 Billing System
- Create a bill
- Add bill items
- View all bills
- View bill details
- Mark bill as PAID
- Revenue summary reports
---
## 􍠽􍳂 Technology Used
- Python
- SQLite3 (built-in database)
- ReportLab (for PDF generation if needed)
---
## ▶􏸏 How to Run the Project
### **1. Install Python**
Download Python from:
https://www.python.org/downloads/
### **2. Install Required Libraries**
Open terminal / CMD and run:
```
pip install reportlab
```
SQLite is already included with Python.
### **3. Run the Program**
```
python hospital.py
```
